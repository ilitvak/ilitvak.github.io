BlocJams

BlocJams is a music application that allows the user to play music and use interactive scroll bars and volume controls. This was the first project I had to build as part of my 12-week immersive bootcamp.


There is a bonus hidden feature on Bloc-Jams. All you have to do is mouse click on 'Turn the Music Up'.


Bloc-Jams was built using HTML5, CSS3, Sass and the AngularJS.

Pomodoro

The Pomodoro Timer is a productivity application that helps users use their time efficiently. You set the time for 25 minutes and during that time you focus on one sole task. I have personally used this timer for my daily activities and can confidently say I feel more productive. Hopefully you have similar results! Pomodoro was built using HTML5, CSS3, Sass and the AngularJS.
