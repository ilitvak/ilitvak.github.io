BodyByYou

My Final Project for Bloc to graduate their Front End Development course.

A nutrition/diet website meant for educating users on the various shopping and eating options. This application utilzies an NutritionIX API that allows users to search for their favorite foods and a search query will return with the nutritional values. This website was created using HTML5, CSS3/Sass, AngularJS, NutritionIX API, and Firebase.io.

Pomodoro-Timer

A Pomodoro-Technique inspired application

The Pomodoro Timer is a productivity application that helps users use their time efficiently. You set the time for 25 minutes and during that time you focus on sole task. This application was made using HTML5, CSS3, and AngularJS.
